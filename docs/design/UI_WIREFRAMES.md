## Navigation Structure

### Sidebar (Store Level)

```
┌─────────────────────────┐
│                         │
│  [Logo] MeeLike         │
│                         │
│  ─────────────────────  │
│                         │
│  🏠 Dashboard           │  → /seller
│  📦 ออเดอร์              │  → /seller/orders
│  🛒 บริการ               │  → /seller/services
│  🏪 ร้านค้า              │  → /seller/store
│  💰 การเงิน              │  → /seller/finance
│  ⚙️ ตั้งค่า              │  → /seller/settings
│                         │
│  ─────────────────────  │
│                         │
│  👥 จัดการทีม           │  → /seller/team
│                         │
│  ─────────────────────  │
│                         │
│  📖 คู่มือ               │
│  💬 ติดต่อเรา            │
│                         │
└─────────────────────────┘
```

### Tabs (Team Level)

```
เมื่อเข้า /seller/team/[id] จะเห็น Tabs:

┌───────┬─────────┬──────┬──────────┬──────────┬─────────┐
│ 📊    │ 👥      │ 📋   │ ✅       │ 💰       │ ⚙️      │
│ภาพรวม│ สมาชิก   │ งาน  │ ตรวจงาน  │ จ่ายเงิน  │ ตั้งค่า  │
└───────┴─────────┴──────┴──────────┴──────────┴─────────┘
   │        │        │        │          │         │
   ▼        ▼        ▼        ▼          ▼         ▼
/[id]   /members  /jobs  /review   /payouts  /settings
```

### Breadcrumb Pattern

```
/seller/team/team-1/jobs
```

## Responsive Breakpoints

### Desktop (≥1024px)
- Sidebar แสดงเต็ม
- Content area กว้าง
- Cards แสดง 3-4 คอลัมน์

### Tablet (768px - 1023px)
- Sidebar collapsed (icons only)
- Content area ปรับตาม
- Cards แสดง 2 คอลัมน์

### Mobile (<768px)
- Sidebar เป็น Drawer
- Bottom navigation สำหรับ Worker
- Cards แสดง 1 คอลัมน์
- Full-screen modals
