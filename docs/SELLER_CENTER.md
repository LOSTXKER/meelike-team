# 🏪 Seller Center

## 📋 Order vs Job (ความแตกต่าง)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  📦 Order = ออเดอร์จากลูกค้า                                                │
│  ├── Buyer สั่งซื้อจากหน้าร้าน                                              │
│  ├── มีหลายรายการได้ (Bot + คนจริง)                                         │
│  └── Seller สร้างเองก็ได้ (รับงานนอกระบบ)                                   │
│                                                                              │
│  📋 Job = งานที่มอบหมายให้ทีม                                                │
│  ├── เฉพาะรายการ "คนจริง" ใน Order                                         │
│  ├── Worker รับงาน → ทำ → ส่ง → Seller ตรวจ                                │
│  └── Progress ของ Job จะอัปเดต Order อัตโนมัติ                              │
│                                                                              │
│  ═══════════════════════════════════════════════════════════════════════    │
│                                                                              │
│  🔄 Flow:                                                                   │
│                                                                              │
│  ลูกค้าสั่ง Order → Seller รับ Order → แบ่งรายการ:                          │
│                                                                              │
│  ├── 🤖 Bot → ส่งไป MeeLike API (อัตโนมัติ)                                │
│  │           Progress อัปเดตเอง                                             │
│  │                                                                          │
│  └── 👥 คนจริง → กดปุ่ม "สร้างงานให้ทีม" ในหน้า Order Detail               │
│                 → Worker รับงาน → ทำ → ส่ง                                  │
│                 → Seller ตรวจ Approve → Progress อัปเดต                     │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
