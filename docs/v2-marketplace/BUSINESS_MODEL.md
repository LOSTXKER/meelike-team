## Anti-Unlike System

### ปัญหา Unlike และวิธีแก้

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  😈 ปัญหา:                                                                  │
│  Worker กด Like → ได้เงิน → Unlike ทันที → Employer เสียเงินฟรี            │
│                                                                              │
│  ✅ วิธีแก้: Screenshot + Freeze + Auto-Close                               │
│  Worker กด Like → Freeze จอ → แคป Screenshot → ปิด Tab อัตโนมัติ           │
│  → Worker ไม่มีโอกาส Unlike เลย!                                           │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Flow การทำงาน

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  📸 Screenshot + AI Verify + Auto Close:                                    │
│                                                                              │
│  ┌─────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐       │
│  │ กด  │ -> │ Freeze  │ -> │ แคปจอ  │ -> │ DOM/AI  │ -> │ ปิด Tab │       │
│  │Like │    │  หน้าจอ │    │ ทันที  │    │ Verify  │    │ อัตโนมัติ│       │
│  └─────┘    └─────────┘    └─────────┘    └─────────┘    └─────────┘       │
│    │             │              │              │              │             │
│    │        0.3 วิ          0.5 วิ        1-2 วิ         ทันที            │
│                                                                              │
│  รวมเวลา: ~2 วินาที                                                        │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Unlike Rate คาดการณ์

| Protection Layer | ป้องกัน | Unlike Rate หลังใช้ |
|-----------------|---------|-------------------|

| + AI Verify | 4% | ~0.5% |
| **รวมทุกระบบ** | **99%** | **<1%** |
