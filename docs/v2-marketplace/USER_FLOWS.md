### What Public Can See vs. What's Hidden

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  [eye] Visible to Public:                                                   │
│  ├── Platform (Facebook, Instagram, TikTok, X)                             │
│  ├── Job Type (Like, Follow, Comment, Share)                               │
│  ├── Price per action (เช่น 0.25/like)                                    │
│  ├── Remaining slots (จำนวนที่เหลือ)                                       │
│  └── Job stats (จำนวนงาน, ราคาเฉลี่ย)                                      │
│                                                                              │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                              │
│  [eye-off] Hidden from Public:                                              │
│  ├── Target URL (ลิงค์โพสต์/โปรไฟล์)                                       │
│  ├── Employer information (ชื่อ, ข้อมูลติดต่อ)                             │
│  ├── Job description details (ถ้ามี)                                       │
│  └── Comment templates (สำหรับงาน Comment)                                 │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
