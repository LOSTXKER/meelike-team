## 2. ธุรกิจของเราคืออะไร?

### คำอธิบายสั้น

**MeeLike เป็น Marketplace Platform ที่เชื่อมต่อผู้ประกอบการ (Seller) กับกลุ่มคนทำงาน (Worker) สำหรับงานบริการ Digital Marketing**

### รายละเอียดธุรกิจ

```
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│                      💼 MeeLike Platform                                 │
│           "Marketplace สำหรับ Digital Marketing Services"                │
│                                                                          │
│  ┌──────────────┐      ┌──────────────┐      ┌──────────────┐           │
│  │   Seller     │      │   Platform   │      │   Worker     │           │
│  │  (ผู้ขาย)    │─────▶│   (MeeLike)  │─────▶│  (ผู้รับงาน) │           │
│  └──────────────┘      └──────────────┘      └──────────────┘           │
│         │                     │                     │                    │
│         │                     │                     │                    │
│    สร้างงาน              จัดการ/จับคู่           รับงาน/ส่งงาน          │
│    จ่ายเงิน              หัก Fee               รับค่าตอบแทน             │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

### บริการหลักที่ให้บริการ

| บริการ | รายละเอียด |
|--------|------------|
| **Social Media Engagement** | บริการเพิ่ม Like, Follow, Comment บน Facebook, Instagram, TikTok |
| **Content Interaction** | บริการดู Video, Share Post, Save Post |
| **Review Services** | บริการเขียน Review, Rating บนแพลตฟอร์มต่างๆ |
| **Digital Task Services** | งาน Digital ทั่วไปที่ทำได้ผ่าน Online |

## 4. โมเดลธุรกิจ (Business Model)

### 4.1 แหล่งรายได้หลัก (Revenue Streams)

```
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│  💰 แหล่งรายได้ของ MeeLike Platform:                                     │
│                                                                          │
│  1️⃣ Platform Fee (จาก Seller)                                           │
│     └── 9-12% ของค่าจ้าง Worker (ลดตาม Rank การใช้งาน)                  │
│                                                                          │
│  2️⃣ Subscription Fee (จาก Seller)                                       │
│     └── ฿0 - ฿399/เดือน (ตามแพ็กเกจที่เลือก)                            │
│                                                                          │
│  3️⃣ Withdrawal Fee (จาก Worker)                                         │
│     └── 2.5-5% ของยอดถอน (ลดตาม Level)                                  │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

### 4.2 รายละเอียด Fee Structure

#### Platform Fee (เก็บจาก Seller)

| Seller Rank | ยอดใช้งาน (3 เดือน) | Platform Fee |
|-------------|---------------------|--------------|
| Bronze | < ฿20,000 | 12% |
| Silver | ฿20,000 - ฿50,000 | 11% |
| Gold | ฿50,000 - ฿150,000 | 10% |
| Platinum | > ฿150,000 | 9% |

#### Subscription Plans (เก็บจาก Seller)

| Plan | ราคา/เดือน | จำนวนทีมที่สร้างได้ | Features |
|------|------------|-------------------|----------|
| Free | ฿0 | 2 ทีม | Basic Features |
| Basic | ฿49 | 5 ทีม | + Analytics |
| Pro | ฿99 | 20 ทีม | + API Integration |
| Business | ฿399 | Unlimited | + Custom Domain |

#### Withdrawal Fee (เก็บจาก Worker)

| Worker Level | งานสำเร็จ | Fee Rate |
|--------------|----------|----------|
| Bronze | 0-50 งาน | 5% |
| Silver | 51-200 งาน | 4% |
| Gold | 201-500 งาน | 3.5% |
| Platinum | 501-1,000 งาน | 3% |
| VIP | 1,000+ งาน | 2.5% |

**ขั้นต่ำในการถอน: ฿20**

## 5. Transaction Flow (การไหลของเงิน)

### 5.1 Flow หลัก

```
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│  💳 TRANSACTION FLOW                                                     │
│                                                                          │
│  ┌─────────┐                                                             │
│  │ Seller  │                                                             │
│  └────┬────┘                                                             │
│       │                                                                  │
│       │ (1) เติมเงินเข้า Wallet                                          │
│       │     💳 PromptPay / Bank Transfer / Credit Card                   │
│       ▼                                                                  │
│  ┌─────────────────┐                                                     │
│  │ Platform Wallet │◀─── เงิน Seller (Escrow)                            │
│  │    (MeeLike)    │                                                     │
│  └────┬────────────┘                                                     │
│       │                                                                  │
│       │ (2) สร้างงาน → หักเงินจาก Wallet                                  │
│       │     (ค่าจ้าง + Platform Fee)                                     │
│       │                                                                  │
│       │ (3) Worker ทำงานเสร็จ → เงินเข้า Worker Wallet                   │
│       ▼                                                                  │
│  ┌─────────────────┐                                                     │
│  │ Worker Wallet   │                                                     │
│  └────┬────────────┘                                                     │
│       │                                                                  │
│       │ (4) Worker ถอนเงิน → หัก Withdrawal Fee                          │
│       │     💳 PromptPay / Bank Transfer                                 │
│       ▼                                                                  │
│  ┌─────────┐                                                             │
│  │ Worker  │                                                             │
│  │ Account │                                                             │
│  └─────────┘                                                             │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
```

### 5.2 ตัวอย่างการคำนวณ Transaction

```
ตัวอย่าง: Seller สั่งงาน 1,000 Like ราคา ฿0.22/Like

┌────────────────────────────────────────────────────────────────┐
│                                                                │
│  📥 Seller จ่าย:                                               │
│  ├── ค่าจ้าง Worker: 1,000 × ฿0.20 = ฿200                     │
│  ├── Platform Fee (12%): ฿200 × 12% = ฿24                     │
│  └── รวม: ฿224                                                 │
│                                                                │
│  📤 Worker ได้รับ:                                             │
│  ├── ค่าจ้าง: ฿200                                             │
│  ├── หัก Withdrawal Fee (5%): -฿10                            │
│  └── สุทธิ: ฿190                                               │
│                                                                │
│  💰 Platform ได้รับ:                                           │
│  ├── Platform Fee: ฿24                                        │
│  ├── Withdrawal Fee: ฿10                                      │
│  ├── หัก Payment Gateway Fee (~2%): -฿4.68                    │
│  └── กำไรสุทธิ: ฿29.32                                        │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## 6. Payment Use Cases

### 6.1 ช่องทางรับเงินที่ต้องการ (Inbound)

| ช่องทาง | ผู้ใช้ | ลักษณะการใช้งาน |
|---------|--------|----------------|
| **PromptPay QR** | Seller, Worker | เติมเงินเข้า Wallet |
| **Bank Transfer** | Seller | เติมเงินจำนวนมาก |
| **Credit/Debit Card** | Seller | สะดวก, รวดเร็ว |
| **True Money Wallet** | Worker | เติมเงินจำนวนน้อย |

### 6.2 ช่องทางจ่ายเงินที่ต้องการ (Outbound)

| ช่องทาง | ผู้ใช้ | ลักษณะการใช้งาน |
|---------|--------|----------------|
| **PromptPay** | Worker | ถอนเงินเข้าบัญชี (หลัก) |
| **Bank Transfer** | Worker, Seller | ถอนเงินจำนวนมาก |

### 6.3 คาดการณ์ Transaction

| ประเภท | จำนวน/เดือน | มูลค่าเฉลี่ย | รวม/เดือน |
|--------|-------------|--------------|-----------|
| **Seller Top-up** | 500 รายการ | ฿2,000 | ฿1,000,000 |
| **Worker Withdrawal** | 3,000 รายการ | ฿300 | ฿900,000 |
| **Subscription** | 100 รายการ | ฿100 | ฿10,000 |

**หมายเหตุ:** ตัวเลขเป็นการคาดการณ์สำหรับ Year 1

## 9. ทำไมต้อง MeeLike?

### 9.1 จุดแข็งของแพลตฟอร์ม

| จุดแข็ง | รายละเอียด |
|---------|------------|
| **Rank-based Fee** | ยิ่งใช้เยอะ Fee ยิ่งลด สร้าง Loyalty |
| **Free to Start** | ไม่มี Barrier เริ่มใช้ได้เลย |
| **Mobile-First** | รองรับการใช้งานบน Mobile 100% |
| **Real-time Dashboard** | Seller/Worker เห็นข้อมูลทันที |
| **Automated System** | จับคู่งาน, จ่ายเงินอัตโนมัติ |

### 9.2 Market Opportunity

- ตลาด Digital Marketing ในไทยโตต่อเนื่อง 15-20%/ปี
- SMEs ต้องการบริการราคาประหยัด
- แรงงาน Gig Economy เติบโตอย่างรวดเร็ว
- ยังไม่มี Platform หลักที่ครองตลาดนี้
