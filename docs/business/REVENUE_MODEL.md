## Platform Fee (Rank-based)

### 🏆 Seller Rank System

| Rank | ยอดจ้าง (Rolling 3 เดือน) | Platform Fee | ≈ จำนวนทีม |
|------|--------------------------|-------------|-----------|
| 🥉 **Bronze** | < ฿20,000 | **12%** | < 1 ทีม |
| 🥈 **Silver** | ฿20,000 - ฿50,000 | **11%** | 1-2 ทีม |
| 🥇 **Gold** | ฿50,000 - ฿150,000 | **10%** | 2-7 ทีม |
| 💎 **Platinum** | > ฿150,000 | **9%** | 7+ ทีม |

### 🔄 Rolling Average 3 เดือน

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  📊 วิธีคำนวณ Rank:                                                     │
│                                                                         │
│  Rank = ยอดเฉลี่ย 3 เดือนล่าสุด                                         │
│       = (เดือนนี้ + เดือนก่อน + 2 เดือนก่อน) ÷ 3                        │
│                                                                         │
│  ✅ ข้อดี:                                                              │
│  • ขึ้น Rank ค่อยๆ ไป ไม่กระโดด                                         │
│  • ตก Rank ช้า เดือนแย่ 1 เดือนไม่ตกทันที                               │
│  • สร้าง Loyalty แต่ยัง Fair                                            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 📈 ตัวอย่าง Rolling Average

| เดือน | ยอดเดือนนั้น | Rolling Avg | Rank |
|-------|-------------|-------------|------|
| เดือน 1 | ฿15K | ฿15K | 🥉 Bronze |
| เดือน 2 | ฿25K | ฿20K | 🥈 Silver |
| เดือน 3 | ฿35K | ฿25K | 🥈 Silver |
| เดือน 4 | ฿60K | ฿40K | 🥈 Silver |
| เดือน 5 | ฿80K | ฿58K | 🥇 Gold! |
| เดือน 6 | ฿100K | ฿80K | 🥇 Gold |

### 📊 ตัวอย่างการคำนวณ

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  📊 ตัวอย่าง: ค่าจ้าง Worker ฿0.20 ต่องาน                               │
│                                                                         │
│  🥉 Bronze (12%):                                                      │
│     ค่าจ้าง ฿0.20 + Fee ฿0.024 = Seller จ่าย ฿0.224                    │
│                                                                         │
│  🥈 Silver (11%):                                                      │
│     ค่าจ้าง ฿0.20 + Fee ฿0.022 = Seller จ่าย ฿0.222                    │
│                                                                         │
│  🥇 Gold (10%):                                                        │
│     ค่าจ้าง ฿0.20 + Fee ฿0.020 = Seller จ่าย ฿0.220                    │
│                                                                         │
│  💎 Platinum (9%):                                                     │
│     ค่าจ้าง ฿0.20 + Fee ฿0.018 = Seller จ่าย ฿0.218                    │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 💰 Platform Fee Revenue ต่อ Rank

| Rank | ยอดจ้างเฉลี่ย | Fee % | Platform ได้/เดือน |
|------|-------------|-------|-------------------|
| 🥉 Bronze | ฿10,000 | 12% | ฿1,200 |
| 🥈 Silver | ฿35,000 | 11% | ฿3,850 |
| 🥇 Gold | ฿100,000 | 10% | ฿10,000 |
| 💎 Platinum | ฿250,000 | 9% | ฿22,500 |

### 🎯 ข้อดีของ Rank-based Fee

1. **ไม่มี Barrier** - เริ่มใช้ได้เลย ไม่ต้องจ่ายล่วงหน้า
2. **Fair** - จ่ายตามการใช้งานจริง
3. **Gamification** - Seller อยากทำยอดเพื่อขึ้น Rank
4. **Lock-in** - ยิ่งใช้เยอะ fee ยิ่งถูก → ไม่อยากย้าย
5. **Word of Mouth** - "ใช้ MeeLike นานๆ fee ลดเอง"
